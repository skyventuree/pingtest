Pingtest

Lightweight and simple URL ping tester for local network

Type the URL into this box and press Add, after added all the URL, press Start!

Ping will be evaluated on this machine/device you are using.

Tip: If you plan on testing multiple URLs another time, you should make a list of them, seperated by comma for future use!

URL Status Time (ms) Remove URL?

Note: Due to CORS, most of the site will show the status code as 0 for external URLs. This is normal.

If the URL failed to respond within 5 seconds, it will be considered as a timeout.
